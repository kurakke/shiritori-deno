# [ポケモンしりとり](https://kurakke-shiritori-deno.deno.dev/)
ポケモンの単語でしりとり出来るウェブサイト
>インターンの応募で提出する作品として作成. 条件は「Denoを用いたしりとりアプリの制作」
## Build With
- Deno
- Deno Deploy
- React
- Type Script
- Styled-components
